About Land Partitioning

Web Application

Land partitioning application is a semi-automated land partitioning application that uses vector data to generate layouts for lot parcel of any shape. Partitioning layouts generated from this application is not intended to replace layouts produced by a surveyor based on field measurements. This tool should instead be used for initial exploratory work prior to actual field survey. Here the how to get started and partitioning your lot parcel.
