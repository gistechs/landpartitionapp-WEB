How it works?


1. Design Reserve Area

Design reserve area using external software platform such as Autocad and ArcGIS. Note: Please ensure the boundary of reserve area extend outside the lot parcel


2. Prepare Data

Prepare two important dataset; lot parcel and reserve area in shapefile format. Compile the shapefile in zip files respectively. Refer to sample data provided below for more information


3. Define Coordinate

Select coordinate system based on the inputted file.


4. Upload Data and Choose Layout

Upload zip file and choose layout from dropdown menu. User can create new layout by navigate to create layout page. Click submit to show layout on the map.
